# 学生プラザ3F 留学交流グループ シフト調整システム

静的なフロントエンドだけで学生のシフト提出と管理者向け集計ができるツールです。ブラウザの LocalStorage にデータを保存するため、テストやデモを行う際は同じブラウザで操作してください。

## 特徴

- **学生向けシフト入力フォーム**: 名前と対象月を選択すると、平日のみの行が生成され、午前/午後/1日/その他(任意の時間帯)を選択できます。
- **その他の時間帯**: ドロップダウンで 30 分刻みの時刻を選び、自由に時間を指定可能です。
- **特別日表示**: 広島大学の試験日や授業振替日など、事前に登録した日付はハイライトされ、備考にも表示されます。
- **Admin 集計ビュー**: 提出された全員分のデータを日付×勤務帯で集計し、Excel 風の表で確認できます。名前フィルタや月変更にも対応しています。

## 使い方

1. 任意の HTTP サーバーでルートディレクトリを公開します。例: `python -m http.server 8000`。
2. ブラウザで `http://localhost:8000` を開きます。
3. 学生は「学生用シフト入力」で名前・対象月を選び、各平日でシフトを選択して提出します。提出後は LocalStorage に保存され、同じブラウザで Admin 画面に即時反映されます。
4. 管理者は「Admin シフト集計」で対象月や表示する名前を切り替えながらシフト表を確認できます。「シフト表を更新」ボタンで再描画も可能です。

## データ永続化について

- データは `localStorage` の `pa-shifts` キーに JSON 形式で保存されます。
- 別のブラウザやシークレットウィンドウではデータが共有されないため、実運用ではサーバー側の保存ロジックに置き換えてください。

## 開発メモ

- `SPECIAL_DAYS` (script.js) に特別日を追記することでカレンダーへの表示を増やせます。
- `data/classReplacementDays.js` に momiji カレンダー（PDF）から転記した授業振替日をまとめています。毎年度更新してください。
- スタッフ一覧は `NAMES` 配列で管理しています。増減したい場合は配列を書き換えてください。
